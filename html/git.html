Git

  • git init — инициализация нового Git-репозитория в текущей папке.
  • git status — показывает статус файлов в репозитории: изменённые, неотслеживаемые и т.д.
  • git commit -m "комментарий" — создаёт коммит с описанием изменений.
  • git log — показывает историю коммитов.
  • git branch имя_новой_ветки — создание новой ветки.
  • git clone URL — клонирование удалённого репозитория на локальный компьютер.
  • git push — отправить изменения на удалённый репозиторий.
